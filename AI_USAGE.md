# AI Usage 

## Which AI tools were used?
The LLMs used in this project were OpenAI's ChatGPT, Github Copilot for students, and Anthropic's Claude. 

## How were they used?
ChatGPT and Copilot were used as a baseline for vibe-coding and pair programming. For the water intake feature, they entirely depended on to write code and debug. Claude was used to further debug issues with the water intake feature. ChatGPT and Copilot were used as pair programmers for the food intake tracker. This means that I wrote initial code until I needed help or review for my work. This is where ChatGPT and Copilot provided suggestions for improvement in code and UI.

## Were there any limitations?
During the vibe-coding experiment, ChatGPT ran out of tokens after enough prompting and Claude served as a fresh "pair of eyes" for the project. Copilot was lacking in quality in its chat functionality compared to its useful inline code completion feature.
